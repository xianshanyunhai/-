# 数学课前三分钟抽取系统

一个现代化的HTML5应用，用于按日历的随机人员抽取。

## ✨ 主要功能

- 📅 **智能日历**：显示5个月的日历视图，支持日期选择和分配
- 🎯 **随机抽取**：一键随机分配人员到选中的日期
- 👥 **人员管理**：动态添加/删除人员，支持列表间移动
- 💾 **数据持久化**：自动保存到本地存储，支持导入导出
- 📊 **实时统计**：显示抽取进度和分配情况
- 🎨 **现代化UI**：扁平化设计，响应式布局

## 🚀 使用方法

1. **打开应用**：在浏览器中打开 `index.html`
2. **选择日期**：在日历中点击要抽取的日期
3. **随机抽取**：点击"一键抽取"按钮进行批量分配
4. **管理人员**：使用底部按钮添加/删除人员
5. **数据管理**：导出/导入数据，或重置所有设置

## ⚠️ 重要操作

- **清除分配**：将所有已抽取人员移回未抽取列表
- **重置数据**：⚠️ 删除所有数据，恢复初始状态
